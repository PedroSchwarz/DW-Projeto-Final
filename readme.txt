Título do Projeto: NOFILTER

Objetivo: Desenvolver o layout das páginas de um sistema que serve como 
uma rede social para fotógrafos, onde nela os profissionais teriam
a possibilidade de publicarem seus trabalhos e de dilvugarem 
suas outras mídias sociais.
Além disso, teriam a opção de comentar e dar notas para o trabalho de outros usários.
Na página inicial seriam expostos os melhores posts e autores da semana.

Justificativa de Desenvolvimento: O projeto foi desenvolvido em cima de outro 
sistema feito por mim, uma rede social para locais de acampamentos. Porém,
com outro tema e que também servirá de um possível aplicativo móvel para 
outras disciplinas ou até mesmo TCC.

Outras informações: Para o plugin foi escolhido o Flickity (https://flickity.metafizzy.co/),
um carousel de imagens. Esse plugin é bem leve e possui um layout agradável e compatível com o do sistema
que foi desenvolvido. O plugin Flickity foi integrado a partir de CDN (link e script).
Para os ícones do sistema foi utilizado a CDN do Font Awesome (https://cdnjs.com/libraries/font-awesome),
um kit com vários ícones.
As imagens presentes no sistema foram pegas do site Unsplash (https://unsplash.com/), um site para 
compartilhamento de imagens gratuitas e de uso livre. Nenhuma delas foram baixadas e colocadas na estrutura 
do projeto para manter o mesmo com um tamanho moderado.
